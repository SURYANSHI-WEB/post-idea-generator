# PostSpark

Instantly generate scroll-stopping social media post ideas and actionable execution steps based on your specific niche. 

## Features

* **Smart Idea Generation:** Creates highly targeted, short-form content ideas tailored for Instagram Reels and YouTube Shorts.

* **Actionable Scripts:** Automatically breaks down each idea into a practical, 5-step execution plan.

* **Export to PDF:** One-click functionality to download your generated ideas into a cleanly formatted document.

* **Modern Interface:** Sleek, animated UI featuring a dark mode aesthetic with neon pink and purple gradients.

* **Secure Architecture:** Utilizes a serverless proxy to ensure API keys remain hidden and secure from the client side.

## Tech Stack

* **Frontend:** React, Tailwind CSS, Framer Motion

* **Backend / API:** Vercel Serverless Functions

* **Integrations:** OpenRouter API (Anthropic Claude-3-Haiku), Axios, jsPDF

## Usage

*Enter your specific niche or topic (e.g., "fitness tips for beginners", "startup finance").*

*Click Generate Ideas.*

*Review the generated hooks and step-by-step instructions.*

*Click Download as PDF to save your content strategy offline.*
